EthiCapital

An ESG based, investment platform.

I built EthiCapital as a platform for both Investors and Financial Advisors to meet and start a financial relationship. An Investor can fill out a questionnaire and EthiCapital will select an investment model that matches their risk tolerance. All investments are based on actual ESG stocks and bonds. (ESG stands for Environmental, Social, and Goverance... basically ethical investments). I designed EthiCapital as my Actualize Capstone Project.

EthiCapital is largely built on:

  • Ruby on Rails
  • PostgreSQL
  • Google Maps API
  • Google Charts
  • Wicked_pdf
  • CarrierWave

This site is currently deployed on Heroku at: http://ethicapital.herokuapp.com

My next goal with EthiCapital is to deploy it on AWS using Docker to resolve deployment bugs and reduce loadtimes.
